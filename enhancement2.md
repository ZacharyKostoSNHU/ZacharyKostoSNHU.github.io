## Artifact
**Event Tracker Mobile Application**
Originally developed in Android Studio using Java during CS 360 Mobile Architecture and Programming.


## Overview
This enhancement focuses on improving how event data is managed within the application by applying appropriate data structures and algorithmic techniques. The goal was to improve performance, prevent duplicate entries, and enhance the overall user experience.


## What Was Improved
- Introduced a HashSet to prevent duplicate events.
- Improved lookup efficiency when checking for existing events.
- Implemented sorting using Collections.sort() to organize events alphabetically.
- Improved overall data handling and performance.
